## 四、演示约束（500MB 服务器）

| 场景 | 风险 | 对策 |
|------|------|------|
| 知识库上传 + 向量索引 | 加载模型冲到 400MB+ | 演示前预热：先上传一个文件 |

| 多人并发 | 每个 SSE 会话 ~1-5MB | 演示时一人操作 |
| 其他 | 🟢 安全 | 外部 API 或简单 DB 操作 |

| 配置 | 值 |
|------|-----|
| Uvicorn workers | 1 |
| 文件上传限制 | ≤5MB |
| sentence-transformers | 懒加载 |

2. 点击"游客试用" → 直接进入聊天
3. 游客点击"知识库" → 弹窗"注册后可用"
4. 注册 → 自动登录，游客会话迁移到账号
```

### 主界面（1 分钟）

```
5. 看到四入口：🧑个人模式 / 👥团队模式 / 📚知识库 / ⚙️模型管理
   每个卡片有状态摘要（会话数/文件数/模型名）
```

### 个人聊天（3 分钟）

```
6. 发消息"写一个快速排序"
   → SSE 流式，右侧栏显示当前 Agent
   → 思维链展开：Planner → Coder → Executor → Tester
   → 代码块语法高亮，一键复制
7. 点思维链折叠/展开 Agent 卡片
```

### Agent 开关（1 分钟）

```
8. 右侧栏关闭 Tester → 发送新消息 → 跳过 Tester
9. 会话信息 Tab → 显示 Token/耗时 → 导出报告 .md
```

### 编排画布（2 分钟）

```
10. 切换到编排页 → React Flow 画布
11. 拖入新节点 → 连线 → 保存
```

### 模板市场（1 分钟）

```
12. 切换到模板市场 → 选择一个模板 → 创建工作空间 → 创建项目
```

### 团队模式（3 分钟）

```
13. 创建组织 → 生成邀请码
14. 另一窗口加入 → 两个窗口实时同步消息
15. @agent 总结今天讨论 → 生成总结
16. @agent 创建待办: 修复bug @B → 勾选完成 → 系统通知
```

### 知识库 + 模型管理（2 分钟）

```
17. 知识库页面上传文件 → 列表刷新
18. 模型管理页面显示当前模型 + 角色映射
```

### 收尾

```
全程约 15 分钟，覆盖全部核心功能
```
