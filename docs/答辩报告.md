## 一、项目概述

### 1.1 项目简介

本项目构建了一个**由 8 个专业 Agent 组成的多智能体协作平台**：Planner（规划）、Retriever（检索）、Coder（编程）、Writer（写作）、Tester（评审）、Summarizer（总结）、Bot（对话）、Executor（执行）。

用户用自然语言描述需求，系统通过 **Router 意图分类器**自动识别任务类型和复杂度，由 **LangGraph 状态图**编排多个 Agent 按流程协作，通过 **SSE 流式协议**实时输出每个 Agent 的执行过程。

### 1.2 核心能力一览

| 能力维度 | 实现方式 |
|----------|----------|
| **多角色协作** | 8 个专业 Agent，LangGraph StateGraph 统一编排 |
| **意图识别** | Router LLM 自动分类 5 种任务类型 + 轻/重复杂度 + 关键词覆写安全网 |
| **任务拆解** | Planner Agent 自动将复杂任务拆解为可执行步骤清单 |
| **知识检索** | ChromaDB 向量数据库 + BAAI/bge-small-zh-v1.5，按用户物理隔离 |
| **代码执行** | Coder Agent 编写 Python 代码 + Executor 执行验证 |
| **可视化** | Pillow 原生绘图（柱状图/折线图）+ OCR 图片文字提取 |
| **网络搜索** | DuckDuckGo 免费搜索，Agent 可实时获取最新信息 |
| **流式输出** | SSE 协议实时推送，agent_start/token/agent_end/done 事件 |
| **拖拽编排** | React Flow DAG 画布，可视化设计 Agent 流水线 |
| **团队协作** | 组织管理 + 频道聊天 + @agent 智能命令 + 待办系统 |

### 4.2 Agent 协作流程

```
用户输入 → Router 分类 → Planner 拆解任务
    ↓
LangGraph StateGraph 编译图
    ↓
┌──────────────────────────────────────────┐
│  Planner → Retriever → Coder → Tester   │
│                  ↓          ↓            │
│               Writer    Executor         │
│                  ↓          ↓            │
│              Summarizer ←────────────────│
│                  ↓                       │
│                Bot                       │
└──────────────────────────────────────────┘
    ↓
SSE 流式输出 → 前端逐 token 渲染
```

## 五、核心功能

### 5.1 SSE 流式聊天

- 8 种事件类型：agent_start / token / agent_end / done / error / cancelled
- 每 Agent 携带 elapsed_ms 和 token_count
- 思维链面板：同 Agent 多次执行不覆盖（name\x00N 唯一 key）
- 自动保存：流式完成后 500ms 防抖持久化

### 5.2 React Flow 编排画布

- 拖拽创建节点（Agent / Router / Start）
- 自由连线构建 DAG
- Router 节点双击编辑条件分支
- 保存后反映到 agent-config，后端动态编译 LangGraph

### 5.3 团队协作

- 6 位邀请码加入组织
- 多频道聊天 + SSE 实时推送
- @agent 命令：总结讨论 / 创建待办 / 搜索知识库
- 待办面板：创建/分配/完成

### 5.4 游客模式

- 免认证直接使用（`POST /api/chat/guest`）
- 受限功能弹窗提示注册
- 注册后自动迁移游客会话

### 5.5 网络搜索

- DuckDuckGo 免费 API，Agent 可实时搜索
- Planner 和 Bot 自动调用 web_search 工具

## 六、关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| Agent 框架 | LangGraph | 状态图天然支持条件路由和多分支，比 AutoGen 更灵活 |
| 前端框架 | React + daisyUI | 组件生态丰富，Tailwind 主题一致性好 |
| 流式协议 | SSE | 比 WebSocket 简单，单向推送足够 |
| 数据库 | SQLite + WAL | 零配置，单文件，Render 免费实例友好 |
| 向量库 | ChromaDB | 轻量级，纯 Python，无需额外服务 |
| 代码执行 | 子进程 | 简单可靠，Docker 沙箱开销大不适合 500MB 实例 |
| 搜索工具 | DuckDuckGo | 免费无 Key，演示足够 |
| 部署 | Cloudflare Pages + Render | 免费额度，全球 CDN |
