## 一、项目概述

本项目构建了一个**由 8 个专业 Agent 组成的多智能体协作平台**，涵盖规划、检索、编程、写作、测试、总结、对话、执行全流程。用户用自然语言描述需求，系统自动识别意图、拆解任务、编排 Agent 协作，通过 SSE 流式实时输出结果。

### 核心能力

| 维度 | 实现 |
|------|------|
| **多角色协作** | 8 Agent，LangGraph StateGraph 编排 |
| **意图识别** | Router LLM 5 种任务类型 + 复杂度分级 |
| **知识检索** | ChromaDB + BGE 嵌入，按用户隔离 |
| **代码执行** | Python 沙箱 + Tester 评审 |
| **可视化** | Pillow 绘图 + OCR 识别 |
| **网络搜索** | DuckDuckGo 免费实时搜索 |
| **流式输出** | SSE 协议，逐 token 实时推送 |
| **拖拽编排** | React Flow DAG 画布 |
| **团队协作** | 组织管理 + 频道聊天 + @agent + 待办 |

## 四、系统架构

```
前端 (React 19 + daisyUI 5)
  ├── 聊天页 (SSE 流式)
  ├── 编排画布 (React Flow)
  ├── 监控评估 (Recharts)
  ├── 团队协作 (频道 + 待办)
  └── 游客模式 (免认证)
         │ REST + SSE
后端 (FastAPI)
  ├── 聊天管道 (router/)
  ├── 工作空间 + 项目 (workspace/)
  ├── 组织 + 团队聊天 (organizations.py + team_chat.py)
  ├── 知识库 + OCR (app/)
  ├── 认证 + 权限 (user/)
  └── SQLite v5 (13 表)
```

### Agent 协作流程

```
用户输入 → Router 分类 → Planner 拆解
    ↓
LangGraph 动态编译
    ↓
Planner → Retriever → Coder → Tester
    ↓          ↓          ↓         ↓
  Bot      Writer    Executor   Summarizer
    ↓
SSE 流式 → 前端渲染
```

## 五、关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| Agent 框架 | LangGraph | 状态图支持条件路由，比 AutoGen 灵活 |
| 前端 | React + daisyUI | 组件生态丰富，Tailwind 一致性好 |
| 流式 | SSE | 单向推送足够，比 WebSocket 简单 |
| 数据库 | SQLite + WAL | 零配置，Render 免费实例友好 |
| 向量库 | ChromaDB | 轻量纯 Python |
| 搜索 | DuckDuckGo | 免费无 Key |
| 部署 | CF Pages + Render | 免费额度 + 全球 CDN |
